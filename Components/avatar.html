Avatar
Avatar is where a user's profile photo is displayed. They used to be available in square shape however they are now mostly circular. Avatars are used in every website. The main purpose of avatar is to provide identity to the user.
Profile Avatar
Profile avatar can be created by using the class avatar, inside an img tag
Preview
Square Avatar
Adding a class 'square' to existing class, will generate a square shaped avatar.
Preview
Size Avatar
Size of the avatars can be changed by adding tiny, medium or large class to the existing avatar class.
